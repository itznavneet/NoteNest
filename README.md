# 📝 NoteNest – Smart Event Noter App

NoteNest is a full-stack web application that allows users to create, manage, and view personal events like birthdays, meetings, reminders, and anniversaries. It's designed to keep track of important events with an intuitive and modern UI.

## 🚀 Features

- 🔐 User Authentication (JWT based)
- 📅 Event creation with:
  - Date
  - Time (with AM/PM support)
  - Type (Birthday, Meeting, etc.)
  - Optional remark
- 🧠 Context API for managing auth stat
- 💾 LocalStorage support
- 📦 Backend: Node.js, Express.js, MongoDB
- 🖥️ Frontend: React.js, CSS modules

## 🛠️ Tech Stack

**Frontend**: React.js, Context API, Axios, Toastify  
**Backend**: Node.js, Express.js, Mongoose  
**Database**: MongoDB  
**Auth**: JWT, bcrypt  
**UI**: Custom CSS
